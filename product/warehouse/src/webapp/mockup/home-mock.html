What is this?

Codekvast collects usage data for your applications.

The collected data helps you identify Truly Dead Code, i.e., code that no-one has used in production for a certain period of time.

By continuously removing truly dead code, you will keep your code base compact. The smaller code base, the faster development cycle.

Codekvast does not modify your code in any way, it just enables you to make informed decisions.
